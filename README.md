# AttendanceAnomalyDetector
Javaで開発した勤怠異常検知システムです。 社員の出退勤データをもとに、遅刻・長時間労働などの異常を自動検知します。 業務システムを想定した構成で、勤怠データ管理・異常判定ロジック・分析処理を分離し、 拡張しやすい設計を意識しています。
---

## 📌 システム概要
本システムは、企業の勤怠管理業務を想定した **異常検知ツール** です。  
出勤・退勤時刻から勤務時間を算出し、以下のような異常を検出します。

- 遅刻（例：9:00以降の出勤）
- 長時間労働（例：10時間以上の勤務）
- 今後追加予定：早退、休憩時間超過、勤務間インターバル不足 など

---

## 🏗 使用技術
- **Java 17**
- **LocalDateTime / Duration** を用いた時間計算
- **オブジェクト指向設計（3クラス構成）**
  - Employee（社員情報）
  - AttendanceRecord（勤怠データ）
  - AttendanceAnalyzer（異常検知ロジック）

---

## 📂 プロジェクト構成
AttendanceAnomalyDetector/
├─ Employee.java              # 社員情報クラス
├─ AttendanceRecord.java      # 勤怠データクラス
├─ AttendanceAnalyzer.java    # 異常検知ロジック
└─ AttendanceAnomalyDetector.java  # メイン処理

---

## 🧩 主な機能

### ✔ 遅刻判定
- 出勤時刻が 9:00 を超えている場合に異常として検出

### ✔ 長時間労働判定
- 勤務時間が 10 時間以上の場合に警告を表示

### ✔ コンソール出力
- 異常が検出された社員IDを表示

---

## 🧪 ダミーデータによる動作例
```text
E001 は遅刻の可能性があります
E002 は長時間労働の可能性があります
``` 

## 💡 制作意図
勤怠管理は企業システムの中でも利用頻度が高く、  
北都システムが扱う業務領域（企業・公共・医療）と親和性が高いため、  
実務を意識したポートフォリオとして開発しました。
